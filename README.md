# Mistique
A fast and easy language similar to python

The main feature of this language is that there are two modes, easy mode, fast mode. Fast mode is a tad more complicated and faster. But easy mode is a tad slower but it's easier to type.
## Fast code example
```
#Main function#
def main() {
    #declaring variables#
    int a = 10; #int#
    flt b = 20.4; #float#
    flt c = 0b10100.01101;
    uns flt d = 0b10100.0110; #can only be positive#
    sig flt e = 0b10100.0110; #can only be negative#
    str f = "Hello World"; #string#
    bl g = true; #bool#
    int h = 0b0101; #binary 10#
    uns int i = 0b0101; #binary 10#
    sig int j = 0b1010; #binary -10#
    int k = 0x4f92; #hexadecimal 20306#
    print("Hello World");
}
```

## Easy code example
```
#Main function
def main():
    #declaring variables
    a = 10 #int
    b = 20.4 #float
    c = 0b10100.01101 #binary with a decimal
    uns d = 0b10100.0110 #binary with a decimal that can only be positive
    sig e = 0b10100.0110 #binary with a decimal that can only be negative
    f = "Hello World" #string
    g = true #bool
    h = 0b0101 #binary 10
    uns i = 0b0101 #binary 10
    sig j = 0b1010 #binary -10
    k = 0x4f92 #hexadecimal 20306
    print("Hello World")
```
